## Перевод терминов

> Когда на странице первый раз встречается термин, давать в скобках оригинальное английское название.

**Service provider** - сервис-провайдер

**Environment** - рабочее окружение, среда выполнения

**Input** - входные данные

**Request** - запрос

**Response** - ответ

**Cookies** - cookies
